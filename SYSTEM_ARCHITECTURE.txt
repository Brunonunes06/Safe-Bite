┌─────────────────────────────────────────────────────────────────────────┐
│                       MÉTODOS DE PAGAMENTO (USE CASES)                  │
└─────────────────────────────────────────────────────────────────────────┘

  PIX - Pagamento Instantâneo
  ├─ Ideal para: Transferências rápidas
  ├─ Tempo: Instantâneo
  ├─ Gera: QR Code + Código PIX
  ├─ Validade: 30 minutos
  └─ Exemplo:
     const pix = await paymentIntegration.generatePIX(9.90)
     → pix.qrCode    (imagem para escanear)
     → pix.pixCode   (texto para copiar)

  BOLETO - Pagamento com Código de Barras
  ├─ Ideal para: Pagamentos em banco/lotérica
  ├─ Tempo: 1-3 dias úteis
  ├─ Gera: Código de barras + Linha digitável
  ├─ Validade: 3 dias
  └─ Exemplo:
     const boleto = await paymentIntegration.generateBoleto(9.90, address)
     → boleto.barcodeNumber    (para copiar)
     → boleto.digitableLine    (alternativa)

  CARTÃO - Pagamento com Débito/Crédito
  ├─ Ideal para: Pagamento online
  ├─ Tempo: Instantâneo
  ├─ Requer: Token Stripe/MercadoPago
  ├─ Suporta: Parcelamento
  └─ Exemplo:
     const card = await paymentIntegration.processCardPayment(token, 9.90, 1)
     → card.status    ('approved' ou 'rejected')
